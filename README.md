# 🎬 AI Story Video Generator
## 📌 Overview
This project converts voice input into a complete storytelling video using AI.
Pipeline:
🎤 Voice → 📝 Text → 📖 Story → 🔊 Narration → 🖼️ Images → 🎬 Video
---
## 🚀 Features
* Voice-based input
* AI story generation
* Text-to-Speech narration
* Dynamic image generation
* Automatic video creation
* Background music support
---
## 🛠️ Technologies Used
* Python
* Transformers (FLAN-T5)
* OpenCV
* gTTS
* SpeechRecognition
* FFmpeg
---
## ▶️ How to Run
```bash
pip install -r requirements.txt
python Story_Teller.py
```
---
## 📂 Output
Final video:
```
final_story.mp4
``
